Tools and Upgrade Priority Guide

Tool upgrade searches are high-value because players want to know where early resources should go. This page gives the ranking framework before exact costs are verified.

Upgrade priority framework

Rank tools by what they unlock: stamina saved per day, new materials, farm expansion, route access, restoration progress, and money generation. Exact rankings must wait for tool costs and mechanics.
