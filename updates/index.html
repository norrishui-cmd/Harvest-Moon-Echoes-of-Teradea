Trust and Maintenance

Updates, Patch Notes, and Corrections

A guide site for an unreleased game needs a public update history so readers can tell which pages are verified, reported, or awaiting launch checks.

Current update status

July 5, 2026: site structure expanded with dedicated guide URLs. Public game details remain limited, so gameplay-specific tables are labeled as templates until official sources or released gameplay can verify them.

Correction policy

  • Every factual claim about release, platform, mechanics, characters, or items should have a source type.
  • Unverified community claims should not appear as guide instructions.
  • Gameplay data should include game version, test date, and whether it was checked on Switch or Switch 2.
  • When patches change values, keep the old value in a changelog only if it helps returning players.

Post-launch update categories

Patch changes

Balance, bug fixes, performance, progression blockers, UI fixes, and platform differences.

Guide corrections

Gift table fixes, material source corrections, quest trigger updates, and route optimization changes.
