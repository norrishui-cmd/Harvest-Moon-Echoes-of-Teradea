World Guide

Map, Exploration, Mist, and Region Guide

Reported details point to open-world exploration, a strange mist, beasts, and villages across Teradea. This page defines the map coverage the site should build as verified data becomes available.

What the map guide should answer

Exploration content should help players decide where to go next, what to bring, and whether a route is worth the stamina or danger. A useful map page connects each region to materials, unlock requirements, hazards, shortcuts, villages, and companion benefits.

Region template

  • Region name, unlock condition, nearest safe return point, and travel time.
  • Gathering nodes, forage items, ore, fish, rare materials, and respawn behavior.
  • Beasts or hazards, recommended tool level, and whether mounts change the route.
  • Village, shop, quest, restoration, or relationship content connected to the region.

Mist and hazard coverage

If the mist is a progression blocker, the guide should treat it as a system: how it is cleared, what it unlocks, whether it returns, and which quests or restoration steps are tied to it. Until launch, those questions stay open and clearly labeled.
