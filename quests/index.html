Quests and Walkthrough Tracker

A quest hub is essential after launch, but pre-launch it should define a spoiler-safe format and connect quests to materials, regions, characters, and unlocks.

Quest guide structure

Quest content should be split into main story, village restoration, character requests, tutorials, seasonal events, and optional exploration tasks. Each quest should show prerequisites, start location, steps, required items, rewards, and next unlock.

Spoiler-safe publishing

  • Use spoiler-light summaries near the top of each page.
  • Hide late-story names or outcomes behind clear headings.
  • Link required items to the item database rather than repeating fragile data everywhere.
  • Mark version-specific quest bugs or blockers in the updates hub.

Launch priority

On launch week, prioritize quests that unlock tools, mounts, villages, new regions, storage, shops, animals, and romance requirements.
